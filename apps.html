Here are a few apps I built recently. As a backend person, I have AI coding assistants to thank for helping me bring the frontends to life.

Some of these apps take user input, for example AIDentify takes an image and detects if it's AI-generated, Redact Docs takes a PDF and redacts PII, while others are already loaded with input data and show only the output, for example RAG Monitor is pre-loaded with user conversations and Web Inspector is pre-loaded with a website for inspection.

Please note that these apps run on low-compute, cost-optimised infrastructure and use lightweight LLMs, so performance is not production-grade.

AIDentify
Launch app → API Docs →

Analyses any image using 10 pixel-level algorithms to determine if it was AI-generated or captured with a real camera.

InputImage file (JPEG, PNG, WebP, BMP, up to 20 MB)
OutputAI probability score (0–100%) with verdict and per-algorithm breakdown
Redact Docs
Launch app →

Scans PDFs for sensitive personal information using AI. Review each finding before approving, then download a clean file with PII physically removed, not just hidden.

InputPDF containing sensitive data
OutputRedacted PDF with PII physically removed
